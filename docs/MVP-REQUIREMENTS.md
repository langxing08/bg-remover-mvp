## 一、产品概述

### 一句话定位

> 一个在浏览器本地运行的图片背景移除工具，用户上传图片后自动去背景，可预览并下载透明背景结果。

### 核心价值

- **即开即用** — 无需注册、无需安装、打开网页就能用
- **隐私安全** — 图片全程在浏览器本地处理，不上传任何服务器
- **零成本** — MVP 阶段无服务器费用，纯静态部署

### 目标用户

- 需要快速去掉图片背景的设计师、运营人员、电商卖家
- 注重隐私不想上传图片到第三方平台的用户
- 需要简单工具偶尔使用的普通用户

## 三、功能需求

### P0 — 核心功能（必须完成）

| # | 功能 | 描述 | 验收标准 |
|---|------|------|---------|
| F1 | **图片上传** | 支持拖拽上传 和 点击文件选择 | 拖拽区域有高亮反馈；点击触发文件选择器 |
| F2 | **图片格式校验** | 仅接受图片类型文件 | 上传非图片文件时静默忽略或提示 |

| F8 | **重置** | 处理完一张后可以继续处理下一张 | 点击"处理下一张"回到上传状态，之前的结果被清除 |

## 四、技术架构概要

### 已确认的技术选型

| 层级 | 技术 | 选型理由 |
|------|------|---------|
| **构建工具** | **Vite** | 极速 HMR，Cloudflare Pages 原生支持 |
| **前端框架** | **Vanilla JS** | MVP 不需要框架，减少体积和复杂度 |
| **背景移除引擎** | **@imgly/background-removal v1.7** | WASM 本地推理，隐私好，质量高 |
| **AI 模型** | **isnet_fp16** | 精度/速度/体积最佳平衡 (~24MB) |
| **引擎切换** | **环境变量 VITE_ENGINE** | 构建时确定，产物体积最小 |
| **部署平台** | **Cloudflare Pages** | 免费静态托管，零配置 |
| **API 代理 (Phase 2)** | **Pages Functions** | 用于未来 remove.bg API Key 保护 |

## 六、验收标准
